# Ultra AI Telegram Auto Reply Bot

Features:
- 24/7 AI Auto Reply (Google Gemini 2.5-flash)
- Offline AI replies
- Friendly, romantic AI personality
- Real human typing simulation
- Control Panel: /on, /off, /status
- First message shows intro text
- Phone login (No Bot Token required)

## Deployment
1. Sign up at [Railway](https://railway.app/)
2. Create a new project and link this repo
3. Set `main.py` as start file
4. Deploy
5. First-time login asks for phone + OTP
6. Session saves automatically → Bot runs 24/7
